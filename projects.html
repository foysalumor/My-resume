A Cluster-Based System for Discovering Similar Text Documents

It’s a AI based web API, which able to find similar text data from a large number of records or text data collection. We used unsupervised learning technique to develop our AI model, also clustered data set to get searching efficiency. We also developed some demo websites bases on our API, which fetch data also able to show relevant data easily by using our API. In this project I learn data clustering, similar data searching, Including AI on Django application, also team work, analytical ability as soft skills
